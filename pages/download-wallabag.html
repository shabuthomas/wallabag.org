Read our documentation to install wallabag or to update it.

Do you need to migrate from wallabag v1? Read this documentation.

You can find our official repository on Docker hub.

docker pull wallabag/wallabag

Thanks to Softaculous, you can easily install wallabag. Create your account on the Softaculous website and install wallabag.
